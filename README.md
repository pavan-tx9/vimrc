# vimrc

Personal vim config. Auto-installs vim-plug and all plugins on first launch.

## Install

**Linux / macOS:**
```bash
bash <(curl -s https://raw.githubusercontent.com/pavan-tx9/vimrc/main/install.sh)
```

**Windows (PowerShell):**
```powershell
iwr -useb https://raw.githubusercontent.com/pavan-tx9/vimrc/main/install.ps1 | iex
```

Installs vim if missing, backs up any existing `.vimrc`, then open `vim` and everything sets itself up.
